# Requirements

- .NET 6 SDK (Download [here](https://dotnet.microsoft.com/download/dotnet/6.0))

# Running

```cmd
cd WebApi
dotnet run
```

or

```cmd
cd WebApi
dotnet watch run
```

# Testing

```cmd
cd Tests
dotnet test
```

# Using

Test API endpoints locally with built-in [Swagger](https://localhost:5555/swagger)
